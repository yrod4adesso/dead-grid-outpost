## Testing
- Unit tests: Vitest. `npm test` -> `vitest run` (non-watch). Config: vitest.config.ts (environment: node, include src/**/*.test.{ts,tsx}, exclude tests/** and *.spec.ts, @ alias -> ./src).
- E2E: Playwright. `npm run test:e2e` -> `playwright test`, testDir ./tests/e2e. Vitest and Playwright are fully separate (different dirs) — do NOT remove Playwright specs or tests/e2e.
- Typecheck: `npx tsc --noEmit` (must be clean). Build: `npm run build`.
- Building-tree prereq reason strings use the raw building id, e.g. "Requires storage Level 2" (lowercase id) — assert case-insensitively in tests.
